IMDb Movie Data Analysis using SQL

About The Project

This project is based on the IMDb movie dataset collected from Kaggle.
The main objective of the project is to perform data analysis using SQL queries on multiple related tables containing movie information, ratings, genres, directors, and actors.

The project helped in understanding relational database concepts and improving SQL query writing skills using real-world data.


---
